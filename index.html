Prime

Given one hour to create Sieve of Eratosthenes to calculate primes up to but not including a given number

Download this project as a .zip file Download this project as a tar.gz file

Author: Tim Foster

Description:

Was given an assignment to implement a Sieve of Eratosthenes for finding primes below a given input from user. The restrictions were that I was to implement it with Python.

The initial implementation (prime.py) was completed within an hour.

The second implementation (primeoop.py) was after I refactored using OOP styles.
